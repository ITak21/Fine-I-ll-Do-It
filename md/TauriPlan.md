# Tauri Plan

## 선택 스택

1. Tauri 2
2. React 18
3. TypeScript
4. SQLite

## 현재 상태

1. 프론트엔드 기본 파일을 생성했다.
2. `src-tauri` 기본 구조를 생성했다.
3. 현재 환경에는 `rustc`, `cargo`가 없어 실제 실행과 의존성 설치는 아직 불가능하다.

## 다음 구현 단계

1. `npm.cmd install`
2. Rust 설치
3. `npm.cmd run tauri dev`
4. 실제 Excel 파일 가져오기 검증
5. 월별 분석 화면 구현

## 실행 준비 문서

1. 실행 절차와 체크리스트는 `md/RunSetup.md`에 정리한다.
